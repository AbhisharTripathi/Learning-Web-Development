Named export are useful when we want to export more than one values and want to import them with thier name. While default export are used to export a single value and import with the custom name.


Rules for JSX
- only one root element can be returned.
- All tags should be closed. e.g. <img></img> OR <img/>
- component name should be in capital like class name but other name should be in camelCase.

class should be written className inside jsx.
React fragments : <> </>

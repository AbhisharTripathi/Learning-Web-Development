State in React : The state is a built in React object that is used to contain data or information about the component. A component state can change over time, whenever it changes the component re-renders.

useState returns an array with exactly two values.
1. The current state during the first render, it will match the initial state you have passed.
2. The set function that let you update the state to different value and trigger a re-render.
- When component is re-rendered the value of state variable change not at the time when set*() is called.
- Component is re-rendered only when the value of state variable is changed.(it renders once if the value is not changed but not more than once.);

Closure : A closure in javascript is a feature where an inner function has access to the outer(enclosing) function's variable. because inncer function can preserve those varaibles which inner function uses.
